**# 🚀 AI Stock Trading Platform**



**A production-grade stock trading simulator with real-time data, ML predictions, and portfolio management.**



**!\[Platform Preview](https://img.shields.io/badge/Status-Live-success)**

**!\[Tech Stack](https://img.shields.io/badge/Tech-React%20%7C%20FastAPI%20%7C%20AI-blue)**



**## ✨ Features**



**### 🤖 AI-Powered Trading**

**- \*\*Advanced ML Predictions\*\*: LSTM-based price forecasting with 75-95% confidence**

**- \*\*Technical Indicators\*\*: RSI, MACD, Bollinger Bands**

**- \*\*Smart Signals\*\*: AI-generated BUY/SELL/HOLD recommendations**

**- \*\*Sentiment Analysis\*\*: News-based market sentiment (coming soon)**



**### 📊 Professional Trading Interface**

**- \*\*Real-time Price Updates\*\*: Live stock data with 30-second refresh**

**- \*\*Advanced Charts\*\*: Interactive candlestick charts with multiple timeframes**

**- \*\*Portfolio Analytics\*\*: Sharpe ratio, volatility, P/L tracking**

**- \*\*100+ Stocks\*\*: Major US stocks (NASDAQ, NYSE)**



**### 💼 Portfolio Management**

**- \*\*Virtual Trading\*\*: Start with $10,000 virtual cash**

**- \*\*Real-time P/L\*\*: Track profits/losses in real-time**

**- \*\*Transaction History\*\*: Complete buy/sell history**

**- \*\*Risk Metrics\*\*: Portfolio risk analysis and optimization**



**### 🎨 Modern UI/UX**

**- \*\*Dark Mode\*\*: Professional trader-friendly interface**

**- \*\*Smooth Animations\*\*: Framer Motion powered transitions**

**- \*\*Toast Notifications\*\*: Real-time trade confirmations**

**- \*\*Mobile Responsive\*\*: Works on all devices**



**## 🛠️ Tech Stack**



**\*\*Frontend:\*\***

**- React 18 + Vite**

**- TailwindCSS (Dark Theme)**

**- Recharts (Advanced Charts)**

**- Framer Motion (Animations)**

**- React Hot Toast (Notifications)**



**\*\*Backend:\*\***

**- Python FastAPI**

**- SQLAlchemy ORM**

**- PostgreSQL / SQLite**

**- yfinance (Real Stock Data)**

**- Pandas + NumPy (Data Processing)**

**- Scikit-learn (ML Models)**



**\*\*AI/ML:\*\***

**- LSTM Price Prediction**

**- Technical Analysis (RSI, MACD, BB)**

**- Portfolio Optimization**

**- Risk Analytics (Sharpe Ratio)**



**## 🚀 Quick Start**



**### Prerequisites**

**- Python 3.12+**

**- Node.js 18+**

**- Git**



**### Backend Setup**

**```bash**

**cd backend**

**python -m venv venv**

**source venv/bin/activate  # Windows: venv\\Scripts\\activate**

**pip install -r requirements.txt**

**uvicorn app.main:app --reload**

**```**



**### Frontend Setup**

**```bash**

**cd frontend**

**npm install**

**npm run dev**

**```**



**### Access**

**- Frontend: http://localhost:5173**

**- Backend API: http://localhost:8000/docs**
